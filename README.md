# Invoice Manager（請求・入金管理システム）

## 概要
業務委託で発生する請求書の発行・入金管理を効率化するためのシステム。

## 使用技術
- Python
- Googleスプレッドシート
- AppSheet
- Excel

## 機能一覧

### 請求データ登録
- Excel請求書から請求データを取得
- Google Sheetsへ自動登録
- 請求書Noによる重複チェックを実施し、二重登録を防止
